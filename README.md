# Vehicle-type-detector  
Проект студентов 3 курса ИИР НГУ

## Описание задачи

Цель данного проекта заключается в проведении анализа датасета с изображениями транспортных средств для выявления закономерностей, распределений и потенциальных аномалий.

Полученные результаты анализа используются для:
- более глубокого понимания структуры данных,
- оценки качества и сбалансированности датасета,
- выявления факторов, которые могут повлиять на обучение модели,
- выбора наиболее подходящих подходов к построению и обучению модели детекции.

В рамках анализа рассматриваются следующие аспекты:
- распределение классов объектов;
- распределение объектов по типам сцен (одиночные и многoобъектные изображения);
- наличие дубликатов изображений;
- распределение классов внутри различных типов сцен;
- взаимосвязи между классами и контекстом изображений;
- потенциальные смещения (bias) в данных;
- а также дополнительные характеристики датасета (размеры объектов, контекст сцены и др.).

Результаты анализа датасета представлены в структурированном виде в данном репозитории и будут дополняться по мере проведения дальнейших исследований.
