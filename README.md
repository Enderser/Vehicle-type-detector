# Vehicle-type-detector
Проект студентов 3 курса ИИР НГУ

## Описание задачи

Доработка детектора для определение принадлежности объекта к одному из перечисленных
типов транспортных средств:

1. Легковые автомобили и микроавтобусы
2. Грузовые автомобили
    - 2.1 Легкие грузовики (двухосные)
    - 2.2 Грузовики
    - 2.3 Грузовики с прицепом (автопоезда/фуры)
    - 2.4 Грузовики с платформой
    - 2.5 Эвакуаторы
3. Автобусы
4. Спец. транспорт
    - 4.1 Скорая помощь
    - 4.2 Пожарная машина
    - 4.3 Машина ГАИ (ГИБДД, ДПС, МВД и т.д.)
5. Мототранспорт
    - 5.1 Двухколесный мотоцикл
    - 5.2 Скутер
    - 5.3 Мопед
6. Квадроциклы
7. Строй техника
    - 7.1 Трактор
    - 7.2 Самосвал
    - 7.3 Автокран
    - 7.4 Экскаватор-погрузчик
    - 7.5 Бетономешалка
8. СИМ
    - 8.1 Самокат
    - 8.2 Моноколесо
    - 8.3 Электровелосипед
9. Велосипеды

Обнаружение происходит в системах видеонаблюдения за дорожной ситуацией: улицы, дворы и парковки.

## Функции решения

1. Обнаружение транспортных средств в видепотоке камер
2. Классификация их типов
3. Внесение данных об обнаружении в базы данных

# Реализация

## Результаты обучения модели
- **Модель:** YOLO
- **Датасет:** приватный датасет на 27 тыс. объектов
- **Кол-во эпох обучения:** 200
- **Размер изображений**: 640x640
- **Аугментации**: В обучении использовалась аугментация с настройками: цветовые искажения (HSV), повороты, сдвиги, масштабирование, mosaic, mixup и copy-paste
## Метрики
- **mAP0.5:** 0.3157
- **mAP0.5-0.95:** 0.2408
- **Precision:** 0.5492
- **Recall:**  0.6896
- **F1:** 0.607
- **Accuracy:** 0.5544
